# ShopEasy – Basic Online Store

A lightweight, fully static online store built with HTML, CSS, and vanilla JavaScript. No frameworks or build tools required — just open `index.html` in any browser.

## Features

- 🛍️ **Product catalogue** with 9 sample products across three categories
- 🔍 **Category filtering** (Electronics, Clothing, Home)
- 🛒 **Shopping cart** sidebar with add / remove and quantity controls
- 💰 **Live total** that updates as you modify the cart
- ✅ **Checkout flow** with a confirmation toast notification
- 📱 **Responsive design** that works on mobile and desktop

## Getting Started

```bash
# Clone the repository
git clone https://github.com/denisseloyizz/Portfolio.git
cd Portfolio

# Open in your browser
open index.html        # macOS
xdg-open index.html    # Linux
start index.html       # Windows
```

No dependencies to install. All assets are self-contained.

## Project Structure

```
├── index.html   – Page markup
├── styles.css   – All styles
```
